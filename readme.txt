== Description ==

This is a highly customizable jQuery Slides plugin. You can set its width, height, pagination and many
other jQuery Slides parameters. You can use it on your post or page via a schortcode `[jQuery Slides]`
or in your template file using php function `<?php if(function_exists('jquery-slides')){ jquery-slides(); } ?>`.

Slides can be added as post type slide from slides menu.

This plugin is based on jQuery Slides by Vijay Kumar .

== Installation ==

1. Download the plugin
2. Upload the `jquery-slides` folder to the `/wp-content/plugins/` folder
3. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots ==

1. Go to slides menu /wp-admin/edit.php?post_type=slide
2. Go to /wp-admin/options-general.php?page=slides_options to configure the slider.
